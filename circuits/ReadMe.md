# Use [template](https://github.com/erhant/circomkit-examples)

install 
```bash
yarn install
```

compile
```bash
yarn compile
```

info
```bash
npx circomkit info aggregator_128
```
setup
```bash
yarn setup
```

prove
```bash
yarn prove
```

verify
```bash
yarn verify
```

contract
```bash
npx circomkit contract aggregator_128
```

copy contracts
```bash
yarn copy-contract
```

test
```bash
yarn test
```
